# GlykemiaTracker v18.2

Moderná a intuitívna Android aplikácia na sledovanie hladiny glykémie, vyvinutá v Jetpack Compose<br> 
v prostredí **Android Studio Panda 4** v úzkej spolupráci s umelou inteligenciou **Gemini**.

## 🚀 Kľúčové funkcie
- **Jednoduchý záznam:** Rýchle zadávanie hodnôt cez vlastný číselník.
- **Inteligentný prst:** Aplikácia navrhuje prst pre ďalší odber, aby sa predišlo preťaženiu kože.
- **Cloud synchronizácia:** Možnosť zdieľania dát v rámci rodiny cez Firebase Firestore.
- **QR Konfigurácia:** Dynamické nastavenie cloudu naskenovaním QR kódu (bezpečné a bez preklepov).
- **PDF Export:** Generovanie profesionálnych reportov pre lekára jedným kliknutím.
- **Analýza dát:** Prehľadné tabuľky a trendy v grafoch.

## 🛠 Ako začať
1. **Súkromný režim:** Aplikácia funguje hneď po inštalácii offline, dáta sa ukladajú len do mobilu.
2. **Režim zdieľania:**
   - Otvorte priložený `index.html` v prehliadači.
   - Vložte údaje z vašej Firebase konzoly.
   - Vygenerujte QR kód.
   - V aplikácii (5-klik na stavové okno) naskenujte tento kód.

## 💻 Technológie
- Kotlin & Jetpack Compose
- Firebase Firestore (Cloud DB)
- CameraX & ML Kit (QR Skener)
- Android PDF Document API
